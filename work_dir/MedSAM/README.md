# MedSAM Weights

Place the MedSAM model weights in this directory.

## Download

Download the pretrained weights from the [MedSAM repository](https://github.com/bowang-lab/MedSAM) and place the `.pth` file here.

The expected file structure is:

```
work_dir/MedSAM/
└── medsam_vit_b.pth
```
